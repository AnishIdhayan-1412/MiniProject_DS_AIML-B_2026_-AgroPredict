# 🌾 AgroPredict - Crop Yield Prediction Using Soil and Weather Data

## Abstract
AgroPredict is a machine learning project that predicts crop yield based on soil nutrients and weather conditions. Agricultural productivity is heavily influenced by environmental and soil factors such as nitrogen (N), phosphorus (P), potassium (K), temperature, and fertilizer usage. This project applies data science techniques to analyze these features and build a predictive model using the Random Forest Regressor algorithm. The dataset was sourced from Kaggle and contains real-world agricultural data. After preprocessing and exploratory data analysis, the model was trained and evaluated, achieving an R² score of 0.9908, indicating highly accurate predictions. The insights from this project can assist farmers and agricultural planners in optimizing crop yield by understanding the impact of soil and weather conditions.

## Problem Statement
Farmers often struggle to predict crop yield accurately due to the complex interplay of soil nutrients and weather conditions. Inaccurate yield estimation leads to poor resource planning and economic losses. This project aims to build a predictive model that estimates crop yield based on measurable soil and weather parameters.
